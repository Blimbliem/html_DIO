O que é HTML?

HTML (HyperText Markup Language) é uma linguagem de marcação utilizada para criar e estruturar o conteúdo de páginas na web. Ele não é uma linguagem de programação, pois não executa lógica ou cálculos, mas sim organiza informações para serem exibidas nos navegadores.

O termo HyperText refere-se à capacidade de criar links que conectam diferentes páginas ou partes de um documento, enquanto Markup Language significa que ele usa marcas (ou tags) para definir a estrutura e o estilo do conteúdo.

O HTML é a base de qualquer página da web. Ele define o que aparecerá na tela e como o conteúdo será estruturado. Junto com CSS e JavaScript, forma o trio principal para o desenvolvimento de sites dinâmicos e interativos.

Estrutura Basica do HTML - TAGS

Quebra da Estrutura HTML:

  1. Declaração do Tipo de Documento (DOCTYPE html)

    2. Informa ao navegador que o documento está usando HTML5. É obrigatório no início de todo arquivo HTML para garantir a compatibilidade.

  2. Tag "html"

    3. É a raíz de todo documento HTML. Todo conteudo da pagina deve estar dentro dessa tag. Nela pode conter atributos, como "lang" para definir o idioma.

  3. Tag "head"

    4. Contém informações e metadados sobre o documento (não são exibidos diretamente na página). Inclui título, codificação de caracteres, links para CSS, scripts e mais.

  4. Tag "body"

    5. Contém o conteúdo visível da página (textos, imagens, links, botões, etc.). Tudo o que aparecerá na tela do usuário fica dentro dessa tag.

Atributos

Atributos são informações adicionais que as tags HTML podem ter para modificar seu comportamento ou aparência. Eles sempre aparecem na tag de abertura e têm o formato nome="valor".

Exemplos:

  1. href:

    2. Define o endereço de um link.

  2. target:

    3. Abre o link em uma nova aba.

  3. src:

    4. Fonte da imagem.

  4. alt:

    5. Texto alternativo para a imagem.

  5. style:

    6. Aplica CSS diretamente na tag

Textos

Em HTML, os textos podem ser estilizados e organizados com diferentes tags.

Exemplos:

  1. Tag "p"

    2. Cria um paragrafo.

  2. Tags h1 ~ h6

    3. Criam títulos e subtítulos(do maior ao menor).

  3. Tag "strong"

    4. Texto em negrito com ênfase

  4. Tag "mark"

    5. Texto destacado(marcado)

Listas

As listas organizam itens em sequência, podendo ser numeradas ou com marcadores.

Exemplos:

  1. Lista ordenada: "ol"

    2. Numerada (1, 2, 3...).

  2. Lista não ordenada: "ul"

    3. Com marcadores (pontos, quadrados...).

  3. Lista de definição: "dl,dt,dd"

    4. Lista de termos e descrições

Links

Os links são usados para navegar entre páginas ou recursos. São criados com a tag "a" e o atributo href para definir o destino.

Exemplos:

  1. Href:

    2. URL para onde o link aponta.

  2. Target=blak

    3. Abre o link em uma nova aba
